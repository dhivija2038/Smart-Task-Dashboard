# Smart Task Dashboard 🌟

**Smart Task Dashboard** is an interactive and responsive task management web application designed to help users organize, manage, and track daily activities efficiently. The platform improves productivity by allowing users to create tasks, assign priorities, monitor progress, and stay motivated through rewards and achievement systems.

The project provides a user-friendly interface where users can manage tasks effectively while tracking performance and productivity goals.

### Features

* Add and manage daily tasks
* Task title, description, and due date support
* Priority levels (Low, Medium, High)
* Category-based task organization
* Task completion tracking
* Reward points system
* Progress bar for task completion
* Achievement levels and badges
* Pin important tasks
* Dark mode support
* Responsive and attractive user interface

### Technologies Used

* HTML5
* CSS3
* JavaScript
* DOM Manipulation
* Local Storage
* Event Handling

### Objective

The main objective of Smart Task Dashboard is to help users improve productivity and task management by providing an organized platform with tracking, progress monitoring, and motivational features.
